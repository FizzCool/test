网络编程:
    1.网络编程三要素
        A:IP
        B:端口号
        C:协议
    2.IP地址
        (1)概述:网络中计算机的唯一标识
                由4个字节的二进制数字组成
                因为记忆麻烦,日常常用点分十进制表示
        (2)IP地址的组成:网络号段+主机号段
        (3)IP地址的分类
            A类:第一号段为网络号段+后三段主机号段--超级号段
                一个网络号段:256*256*256=1637216台主机
            B类:前两个号段为网络号段+后两段主机号段--大型公司,学校,组织
                一个网络号段:256*256=65536台主机
            C类:前三个号段为网络号段+后一段主机号段--家用,中小型公司,组织
                一个网络号段:256台主机
            D类
            E类
        (4)两个常用的DOS命令
            ipconfig  查看本机IP地址
            ping [ip] 测试本机与指定IP地址通信是否有问题
        (5)特殊的IP地址
            172.0.0.1 回环地址(表示本机)
            x.x.x.255 广播地址
            x.x.x.0   网络地址
    3.InetAddress类
        (0)如果一个类没有构造方法
            A:成员全是静态的(Math,Arrays,Collections)
            B:单例模式(Runtime)
            C:类中有静态方法返回该类的对象(InetAddress)
        (1)构造方法
            没有构造方法
            该类提供了静态方法返回该类的一个实例
            public static InetAddress getByName(String host)通过主机名或ip地址获得一个实例
        (2)常用成员方法
            public String getHostAddress()返回 IP 地址字符串（以文本表现形式）。
            public String getHostName()获取此 IP 地址的主机名。
    4.端口号
        (1)概述:正在运行的程序的标识
        (2)有效范围:0--65535,其中0--1024系统使用或保留端口号
    5.协议
        (1)概述:通信的规则
        (2)UDP
            把数据打包
            数据有限制(64k)
            不建立连接
            速度快
            不可靠
        (3)TCP(三次握手协议)
            建立连接通道
            数据无限制
            速度慢
            可靠
    6.Socket
        (1)概述:网络套接字,包含了IP地址+端口号.通常说的Socket编程其实就是网络编程,也叫套接字编程.
        (2)通信原理:
            通信两端都有Socket
            网络通信其实就是Socket间的通信
            数据在两个Socket间通过IO传输
